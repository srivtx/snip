# Step 1: Read a File

The simplest possible starting point. We just read a code file from disk and print it.

## What you'll learn
- Using `fs.readFileSync` to read files
- Using `process.argv` to get command line arguments

## Run it

```bash
node index.js sample.js
```

## What happens
It reads `sample.js` and prints the raw code to your terminal. That's it.
Nothing fancy yet — we're just making sure we can get code from a file into our program.
